Прорамма была разработана для быстрого анализа объемных многолетних таблиц Excel, путём изменения разрешения excel в .txt -> создания динамической копии таблицы Excel, её обработки и вывода необходимых данных разных ТЭЦ в один Excel.

Такая простая программа позволила решить ежегодную задачу, выполняемую 2 недели за 1,5 часа.


В связи с конфиденциальностью данных, требуемых для работы программы, примера вводимой и выводимой базы данных нет.
Можно только ознакомиться с кодом.
